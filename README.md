## Persyaratan

Install dependency terlebih dahulu:

```bash
pip install -r requirements.txt
```

## Fitur Desktop

### Tambah Antrian
Menambahkan pasien ke daftar antrian.

### Lihat Daftar Antrian
Menampilkan seluruh antrian pasien.

### Panggil Antrian
Memanggil pasien menggunakan suara Bahasa Indonesia.

### Reset Antrian
Menghapus seluruh antrian dan mengembalikan nomor antrian ke awal.
